# Adaline Challenge

---

## Prerequisites
- Node.js (v18 or later)
- npm (v9 or later)

---

## Installation

### Client Setup
1. Navigate to the client folder:
   ```bash
   cd client

2. Install dependencies:
   ```bash
   npm install

3. Start the frontend server:
   ```bash
   npm run dev

### Server Setup
1. Navigate to the Server folder:
   ```bash
   cd server

2. Install dependencies:
   ```bash
   npm install

3. Start the backent server:
   ```bash
   npm run start
